<간단한 함수>

    1. .length 는 길이를 구할 때 사용
    2. .indexOf("문자") 는 해당 문자가 앞에 쓰여진 문자열에서 몇번째에 위치하고 있는지를 구할 때 사용
    ex) "code".indexOf("o") 는 1이 출력되고 "code".indexOf("c") 는 0이 출력된다.
    
    
### 알게 된 점
- 만약 중복된 문자가 있다면 맨 앞의 번째 숫자만 출력

ex) "soohyeon".indexOf("o") 는 1을 출력한다는 말
